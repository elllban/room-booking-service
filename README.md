# Room Booking Service

Сервис для бронирования переговорных комнат с авторизацией, расписаниями и генерацией слотов.

## Технологии

- **Go 1.25** — основной язык
- **PostgreSQL** — хранение данных
- **Docker & Docker Compose** — контейнеризация и запуск
- **JWT** — авторизация
- **gorilla/mux** — маршрутизация
- **golangci-lint** — линтинг кода

## Возможности

- Регистрация и получение JWT токена
- **Admin:** создание переговорных, управление расписанием, просмотр всех броней
- **User:** просмотр свободных слотов, создание и отмена своих броней
- Автоматическая генерация 30-минутных слотов на основе расписания
- Тестовое покрытие >40%
- Swagger документация

## Запуск

```bash
# Склонировать репозиторий
git clone https://github.com/elllban/room-booking-service.git
cd room-booking-service

# Запустить через Docker Compose
docker-compose up --build
